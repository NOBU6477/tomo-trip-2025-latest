## 🌐 GitHub Pages設定（無料ホスティング）

### **どちらの方法でも最後にこれをやる**
```
⚙️ 「Settings」タブをクリック
📄 左メニューで「Pages」をクリック
🌿 「Source」で「Deploy from a branch」を選ぶ
🌿 「Branch」で「main」を選ぶ
📁 「Folder」は「/ (root)」のまま
💾 「Save」ボタンを押す
⏰ 5分待つ
🌐 あなたのウェブサイトが公開される！
```
