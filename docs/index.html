TomoTrip（旅友）とは？

観光地だけじゃなく、「地元の人の日常」にもふれながら旅ができるサービスです。 地元在住のガイドさんが、ふだん使っているお店やスポット、ご近所さんしか知らない景色などを案内します。

3つのポイント

Point 1

地元ガイドと直接マッチング

プロフィールやレビューを見ながら、自分に合いそうなガイドさんを選べます。

Point 2

観光地だけじゃない“日常”の体験

ローカルな食堂・カフェ・ビーチ・イベントなど、ガイドさんおすすめのスポットへご案内。

Point 3

小さなお店も一緒に応援

協賛店と連携し、観光客・ガイド・地域のお店がみんな得をする仕組みを目指しています。

ご利用イメージ

観光客の流れ

  1. アプリでエリアと日程をえらぶ
  2. 気になるガイドにチャットで相談
  3. プランを決めて予約・決済
  4. 当日は集合場所でガイドと合流

地元ガイドの流れ

  1. 得意なエリア・体験内容を登録
  2. 予約リクエストに応えてプラン提案
  3. 当日案内し、体験後にレビューをもらう
  4. 報酬はアプリを通じて安全に受け取り

お問い合わせ・協賛店募集

ガイドとして参加したい方、協賛店として登録を検討されている方は、 公式LINEまたはお問い合わせフォームからご連絡ください。
